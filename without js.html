About Me

Devolper & Artist

Hey, mein Name ist Eric. Ich bin 14 Jahre alt, hobby Front-End Developer und Designer. Am liebsten erstelle ich innovative Front-End deisgns oder arbeite an kleinen Spielen. Über die Letzten Jahre habe ich mir diverse Pogramme wie z.B Adobe Photoshop, Illustrator, After Effects, sowie verschiedene Programmiersprachen, also Python, CSS, Java Script und HTML selber beigetracht. Mehr zu meinen selbst geschriebenen Pogramme oder abstrakten Grafiken, findest du weiter unten, bei den Projekten.



    My Skills

    HTML 5

    Auch wenn HTML als die leichteste "Programmiersprache" gilt, bietet HTML viele möglichkeiten und ist also Front-End Dev. ein durchaus sehr wichtiges tool. Denn jede Webside, besteht aus einem HTML Code. Zwar sieht einze kahler Code nicht besonders schön aus und bringt keine Logik mit sich. Jedoch bringt er die Grundlage für weitere Sprachen wie CSS oder Java Script.

    CSS

    Zwar ist CSS keine notwendige "Sprache" beim erstellenen einer Webside, jedoch lässt CSS die Seite besser aussehen. Denn CSS ist eine reine "styling language". D.h sie bringt die Webside nicht zum Funktionieren, sonder macht z.B den Hintergrund blau oder verschiebt einen Knopf. Um zu sehen, wie wichtig CSS trotz seiner nicht verpflichtenden nutzung wie es bei HTML der der Fall ist, ist. Kannst du auf den unteren Knopf drücken. Die komplett geleiche Seite, wir dir dann ohne CSS angezeigt.

    Java Script

    Java Script ist die erste "richtige Programmiersprache" auf dieser Liste. Den Java Script, hat im Vergleich zu den vorigen Sprachen Logic, man kann z.B Bedingungen aufstellen oder Sachen berechnen lass. Mit HTML und CSS geht dies nicht, da sie reine display "Spraches" sind. Auf diese Seite spielt der Js Part keine große Rolle, er ist jediglich für das Verändern der Navbar, die Projekt-Box, die selbst schreibende Schrift und einzelne andere kleine Elemente verantwortlich. Mit dem unteren Knopf, kannst du nun ausprobieren, wie diese Seite ohne Java Script ausiseht/funktioniert.
