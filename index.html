    Java Script

    Java Script ist die erste "richtige Programmiersprache" auf dieser Liste. Den Java Script, hat im Vergleich zu den vorigen Sprachen Logic, man kann z.B Bedingungen aufstellen oder Sachen berechnen lass. Mit HTML und CSS geht dies nicht, da sie reine display "Spraches" sind. Auf diese Seite spielt der Js Part keine große Rolle, er ist jediglich für das Verändern der Navbar, die selbst schreibende Schrift und andere einzelne andere kleine Elemente verantwortlich. Mit dem unteren Knopf, kannst du nun ausprobieren, wie diese Seite ohne Java Script ausiseht/funktioniert.

    Python

    Python hat wie alle anderen oben geannten "Sprachen", nicht mit dem Front-End Development zu tun. Python im Back-End, veringert z.B die Ladezeiten. Zwar ist es wichtig möglichst niedrige ladezeiten zu haben, jedoch ist Python, auf dieser Webside, deutlich unwichtiger, als alle anderen Sprachen. Jedoch kann Python auch für andere Sachen, wie z.B Maschiene Learing eingesetzt werden. Denn Python ist eine sehr vielseitig einsetztbare "Sprache".
