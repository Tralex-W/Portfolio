About Me

Devolper & Artist

Hey, mein Name ist Eric. Ich bin 14 Jahre alt, hobby Front-End Developer und Designer. Am liebsten erstelle ich innovative Front-End deisgns oder arbeite an kleinen Spielen. Über die Letzten Jahre habe ich mir diverse Pogramme wie z.B Adobe Photoshop, Illustrator, After Effects, sowie verschiedene Programmiersprachen, also Python, CSS, Java Script und HTML selber beigetracht.     My Skills

    HTML 5

    Auch wenn HTML als die leichteste "Programmiersprache" gilt, bietet HTML viele möglichkeiten und ist also Front-End Dev. ein durchaus sehr wichtiges tool. Denn jede Webside, besteht aus einem HTML Code. Zwar sieht einze kahler Code nicht besonders schön aus und bringt keine Logik mit sich. Jedoch bringt er die Grundlage für weitere Sprachen wie CSS oder Java Script.

    CSS

    Zwar ist CSS keine notwendige Sprache beim erstellenen einer Webside, jedoch lässt CSS die Seite besser aussehen. Denn CSS ist eine reine "styling language". D.h sie bringt die Webside nicht zum Funktionieren, sonder macht z.B den Hintergrund blau oder verschiebt einen Knopf. Um zu sehen, wie wichtig CSS trotz seiner nicht verpflichtenden nutzung wie es bei HTML der der Fall ist, ist. Kanst du auf den das CSS-Bild klicken. Die komplett geleiche Seite, wir dir dann ohne CSS angezeigt.
